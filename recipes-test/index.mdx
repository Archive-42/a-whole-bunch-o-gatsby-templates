# Install @lekoarts/gatsby-theme-minimal-blog

Adds [@lekoarts/gatsby-theme-minimal-blog](https://github.com/LekoArts/gatsby-themes/tree/master/themes/gatsby-theme-minimal-blog) to your existing Gatsby site. Typography driven, feature-rich blogging theme with minimal aesthetics. Includes tags/categories support and extensive features for code blocks such as live preview, line numbers, and line highlighting.

---

Install the necessary NPM package

<NPMPackage name="@lekoarts/gatsby-theme-minimal-blog" />

---

Add the theme to the gatsby-config.js

<GatsbyPlugin name="@lekoarts/gatsby-theme-minimal-blog" />

---

Add an example page

<File
  path="content/pages/about/index.mdx"
  content="https://raw.githubusercontent.com/lekoarts/recipes-test/master/page.mdx"
/>

---

Add an example blog post

<File
  path="content/posts/example-blog-post/index.mdx"
  content="https://raw.githubusercontent.com/lekoarts/recipes-test/master/post.mdx"
/>

---

Shadow the hero portion of the index page

<GatsbyShadowFile
  theme="@lekoarts/gatsby-theme-minimal-blog"
  path="src/texts/hero.mdx"
/>

---

Shadow the bottom portion of the index page

<GatsbyShadowFile
  theme="@lekoarts/gatsby-theme-minimal-blog"
  path="src/texts/bottom.mdx"
/>

---
