# Style Guide

Theme UI includes an optional package for building style guides based on the [Theme Spec](/theme-spec).

**NOTE:** the style guide package is currently a work-in-progress. See the [related issue](https://github.com/system-ui/theme-ui/issues/59) for more information.

```sh
npm i @theme-ui/style-guide
```

```jsx
<ThemeProvider theme={theme}>
  <ColorPalette />
  <TypeScale />
  <TypeStyle fontFamily="monospace" />
</ThemeProvider>
```
